Pico G2 Socket

Pico G2 Socket - это wireless Socket, спроектированный под шлем PicoG2 для трекинга шлема и приема данных от беспроводных устройств по Antilatency Radio Protocol.

Дизайн для шлема Pico G2

Pico G2 Socket был специально спроектирован под геометрию шлема Pico G2. Pico G2 Socket позиционируется с помощью разъема USB type-C на шлеме и надежно фиксируется на шлеме.

Прием данных от беспроводных устройств по радиопротоколу

В работе Antilatency Radio Protocol Pico G2 Socket выступает в качестве приемника данных от передатчиков. Pico G2 Socket собирает данные от подключенных к нему беспроводных устройств и передает их на шлем по USB вместе с данными своего трекера.

Дополнительный разъем USB type-C Female

Помимо разъема USB type-C Male для подключения к шлему Pico G2 Socket имеет разъем USB type-C Female. При подключении кабеля к разъему USB type-C Female, Pico G2 Socket начинает работать как USB-переходник для зарядки и проброса USB-данных на шлем. Таким образом, единожды подключив Pico G2 Socket к шлему и зафиксировав, Pico G2 Socket уже не нужно будет отключать: зарядка шлема и передача данных по USB для установки приложений будет происходить через указанный разъем USB type-C Female.
Pico G2 Socket работает в 2-х взаимоисключающих режимах: либо в режиме USB-устройства, передающее данные трекинга на шлем, либо в режиме USB-переходника.
