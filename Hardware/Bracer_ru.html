Bracer

Bracer - это wireless Socket для отслеживания рук пользователя, спроектированный для удобного размещения на руке. В отличие от стандартных VR-контроллеров, добавляя 6DoF отслеживание рук, Bracer оставляет руки пользователя свободными для взаимодействия с физическими предметами.

Low latency radio protocol

Bracer для передачи данных использует Antilatency Radio Protocol. Bracer может выступать только в качестве передатчика.
Для приема данных на хосте обязательно наличие приемника (например, HMD Radio Socket). См. Antilatency Radio Protocol

Rechargeable battery

Bracer имеет встроенный заряжаемый аккумулятор емкостью 250 mAh.
Bracer поставляется вместе со специальным модулем для зарядки аккумулятора. Модуль зарядки вставляется в разъем для трекера Alt. Работа Bracer во время зарядки не поддерживается.

Touch поверхность

Bracer имеет в своем составе touch-поверхность, которая при надетом на руку Bracer, располагается на внутренней стороне ладони. Благодаря этой поверхности Bracer может распознавать жесты захвата-отпускания объектов в VR.

Haptic feedback

Bracer имеет встроенный модуль вибро-отклика, благодаря которому пользователь может получать тактильную обратную связь от своих действий. Модуль вибро-отклика управляется из приложения пользователя с помощью API

Designed for hands

Bracer был специально спроектирован для удобного размещения на руках пользователя. Для подстройки под размер ладони пользователя у Bracer есть затягивающийся шнурок с фиксацией.
