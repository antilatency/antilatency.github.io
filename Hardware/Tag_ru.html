Tag

Tag - это беспроводной Socket, который можно закрепить на любом объекте. Так можно отслеживать положение физических предметов, частей тела и кастомных контроллеров, например, макетов оружия. Сам по себе Tag не может определять положение, но он служит передатчиком данных для Alt.

Функциональные особенности

  • Радиопротокол с низкой задержкой
    Tag может выступать только в качестве передатчика. Для обмена данными к Host должен быть подключён приёмник (например, HMD Radio Socket). Для передачи данных Tag использует Antilatency Radio Protocol.
  • Встроенный заряжаемый аккумулятор
    Tag имеет встроенный заряжаемый аккумулятор ёмкостью 250 mAh. Порт Type-C используется только для зарядки аккумулятора. Функциональность Tag полностью доступна во время зарядки аккумулятора, включая передачу данных трекинга с Alt.
  • Компактный дизайн
    Размер Tag 9 x 18 x 66, а вес всего 18. Вы можете закрепить устройство даже на небольшом объекте. Для надёжной фиксации рекомендуем устанавливать Socket с помощью крепёжных отверстий.
