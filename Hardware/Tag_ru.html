Tag

Tag - это беспроводной Socket, который можно закрепить на объекте или на части тела человека для их отслеживания. Он передаёт данные с подключённого к нему Alt. Используя Tag, вы можете отслеживать положение физических предметов и кастомных контроллеров, например, макетов оружия.

Особенности

  • Низкая задержка при передаче данных
    Tag для передачи данных использует Antilatency Radio Protocol. Tag может выступать только в качестве передатчика.
    Для приема данных на хосте обязательно наличие приемника (например, HMD Radio Socket).
  • Встроенный заряжаемый аккумулятор
    Tag имеет встроенный заряжаемый аккумулятор ёмкостью 250 mAh. Порт Type-C используется только для зарядки аккумулятора. Функциональность Tag полностью доступна во время зарядки аккумулятора, включая передачу данных трекинга с Alt.
  • Компактный дизайн
    Габариты Tag (9 x 18 x 66 mm), а вес всего (18 g), что позволяет закрепить устройство даже на небольших объектах. Для надёжной фиксации рекомендуем устанавливать Socket с помощью крепёжных отверстий.(см. TODO файл с моделькой корпуса).
