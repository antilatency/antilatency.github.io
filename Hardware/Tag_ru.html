Tag

Беспроводной Socket, который можно удобно разместить даже на небольшом объекте из-за малых габаритов и веса. Кроме того, Alt имеет встроенный аккумулятор и остаётся полностью функционален во время зарядки.

Низкая задержка при передаче данных

Alt может выступать только в качестве передатчика. Для обмена данными к Host должен быть подключён приёмник (например, HMD Radio Socket). Для передачи данных Alt использует Antilatency Radio Protocol.

Встроенный аккумулятор

Alt имеет встроенный заряжаемый аккумулятор ёмкостью 250 mAh. Порт Type-C используется только для зарядки аккумулятора. Функциональность Alt полностью доступна во время зарядки аккумулятора, включая передачу данных трекинга с Alt.

Компактный дизайн

Размер устройства 16x16x20.5, а вес всего 12. Alt можно закрепить на объекте при помощи двустороннего скотча, но для надёжной фиксации рекомендуем устанавливать Socket с помощью крепёжных отверстий.
