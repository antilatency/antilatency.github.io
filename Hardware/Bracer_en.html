Bracer

The Bracer is a wireless Socket to track the user's hands designed for an ergonomic fit. Unlike standard VR controllers, the Bracer adds on 6DoF hand tracking while keeping the user's hands free to interact with material objects.

Low latency radio protocol

The Bracer uses the Antilatency Radio Protocol to send data. The Bracer can act only as a transmitter.
The host must have a receiver to receive data (for example, a HMD Radio Socket). For more information, please see Antilatency Radio Protocol

Rechargeable battery

The Bracer has a built-in 250 mAh rechargeable battery.
The Bracer is supplied with a battery charger module. It is inserted into the Alt tracker slot. The Bracer will not function while charging.

Touch pad

The Bracer includes a touch pad. When a Bracer is worn on one's hand, this pad fits on the palm. Thanks to this pad, the Bracer can detect grasp and release gestures in VR.

Haptic feedback

The Bracer has a built-in haptic response unit that allows for tactile feedback from user actions. This unit is controlled from a user application with the help of an API

Designed for hands

The Bracer has been custom-designed to be worn on one's hand. The Bracer has a retainer strap that can be fixed in position to fit the size of the user's palm.
