HMD Radio Socket

HMD Radio Socket - это wireless Socket, который поддерживает как прием данных от беспроводных устройств по Antilatency Radio Protocol, так и передачу данных трекинга при размещении на объектах отслеживания.

Универсальный приемник данных по радиопротоколу

В работе Antilatency Radio Protocol HMD Radio Socket выступает, как правило, в качестве приемника данных от передатчиков. USB type-C порт в этом случае используется для питания HMD Radio Socket и для передачи данных на Host по USB.
См. также ниже другие режимы работы HMD Radio Socket.

Работа в качестве передатчика данных

HMD Radio Socket через свойство “Mode” может быть сконфигурирован для работы в качестве передатчика данных по радиопротоколу. В этом случае HMD Radio Socket получает питание от внешней батареи.

Внешняя батарея

К HMD Radio Socket через порт USB type-C может быть подключена внешний Power bank для питания сокета. HMD Radio Socket в этом случае работает в качестве передатчика, аналогично Tag

Поддержка Extension board

HMD Radio Socket поддерживает работу с Antilatency Extension Board. Antilatency Extension Board подключается к HMD Radio Socket через порт USB type-C для передачи данных о внешних триггерах и управлением откликом(вибрация и т.д.). HMD Radio Socket передает эти данные вместе с данными трекинга своему приемнику.
Подробнее см. Antilatency Hardware Extension Interface.

Modular design

HMD Radio Socket может рассматриваться как альтернативу использованию Tag при трекинге объектов, в случае если объект отслеживания обладает собственной батареей или же внешнюю батарею удобно разместить внутри или снаружи корпуса объекта. В этом случае HMD Radio Socket имеет меньший размер, чем Tag, а питание приходит с внешней батареи.
HMD Radio Socket можно закрепить на двухсторонний скотч или используя крепежные отверстия для надежной фиксации (TODO см. файл с моделькой корпуса).
