Tag

Tag - это wireless Socket, который можно закрепить на объекте или части тела человека для их отслеживания.
Tag сам по себе не определяет свое положение, а передает данные с Alt, вставленного в него.
С использованием Tag можно отслеживать физические предметы и кастомные контроллеры, например, макеты оружия.

Low latency radio protocol

Tag для передачи данных использует Antilatency Radio Protocol. Tag может выступать только в качестве передатчика.
Для приема данных на хосте обязательно наличие приемника (например, HMD Radio Socket). См. Antilatency Radio Protocol

Rechargeable battery

Tag имеет встроенный заряжаемый аккумулятор емкостью 250 mAh. Порт Type-C используется только для зарядки аккумулятора. Функциональность Tag полностью доступна во время зарядки аккумулятора, включая передачу данных трекинга с Alt.

Compact design

Tag имеет компактный размер (8.5x18.2x65.7 mm) и маленький вес (18 g), что позволяет закрепить его даже на небольших объектах. Tag можно закрепить на двухсторонний скотч или используя крепежные отверстия для надежной фиксации (см. TODO файл с моделькой корпуса).
