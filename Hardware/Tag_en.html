Tag

The Tag is a wireless Socket that can be attached to an object or a part of the human body to track them.
The Tag does not track its own position, but relays data from the Alt fitted into it.
The Tag can track material objects and custom controllers, for example dummy guns.

Low latency radio protocol

The Tag relies on the Antilatency Radio Protocol to transmit data. The Tag can act only as a transmitter.
In order to receive data the host must have a receiver, for example a HMD Radio Socket. For more information, please see Antilatency Radio Protocol

Rechargeable battery

The Tag has a built-in 250 mAh rechargeable battery. The USB Type-C port is used only to charge the battery. The Tag is fully functional while the battery is charging, including transmission of tracking data from the Alt.

Compact design

The Tag is highly portable (8.5x18.2x65.7 mm) and weighs as little as 18 grams, so it can be attached even to small objects. You can secure the Tag with double-sided adhesive tape or use the mounting holes for a tighter fit (please see the TODO file with a model of its casing).
