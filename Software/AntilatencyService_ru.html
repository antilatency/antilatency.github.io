AntilatencyService

Служебное приложение для обновления прошивки и просмотра дерева подключённых устройств. Кроме этого, AntilatencyService сохраняет настройки зон трекинга и Placement для разных VR устройств.
После установки приложение необходимо запустить, чтобы выполнить первичную настройку устройств и зон трекинга.
AntilatencyService не поддерживает параллельную работу двух и более приложений. Поэтому, прежде чем начать работу с новым приложением, закройте предыдущее.

Environments

Во вкладке Environments можно загрузить, отредактировать или создать зону трекинга.

Placements

Во вкладке Placements можно настроить расположение трекера Antilatency на устройствах для VR. По умолчанию в приложении доступны решения для некоторых популярных моделей шлемов виртуальной реальности.

Device Network

Во вкладке Device Network можно посмотреть системные и пользовательские свойства каждого из подключённых устройств, получить данные о версии прошивки и обновить её при необходимости.
