Antilatency Service

An utility allows configuring the tracking and updating Antilatency devices.
Настроить любое из устройств Antilatency, загрузить или создать зону трекинга, обновить прошивку — всё это легко сделать с Antilatency Service. Приложение доступно на Windows 10 и на Android 6.0+.
После установки приложение необходимо запустить, чтобы выполнить первичную настройку устройств и зон трекинга.
Подробнее про установку можно почитать тут:

Environments

Во вкладке Environments можно загрузить, отредактировать или создать зону трекинга.

Placements

Во вкладке Placements можно настроить расположение устройства Antilatency на шлеме виртуальной реальности. По умолчанию в приложении доступны решения для некоторых популярных моделей.

Device Network

Во вкладке Device Network можно посмотреть системные и настраиваемые характеристики каждого из подключённых устройств, получить данные о версии ПО и обновить её при необходимости.
