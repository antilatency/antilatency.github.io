Hardware

Alt

The Alt is an optical inertial tracking module. The Alt is located on objects being tracked and determines their position in space in relation to infrared markers.
Alt is supplied as a bundle with a wired Socket USB.

Bracer

The Bracer is a wireless Socket to track the user's hands designed for an ergonomic fit. Unlike standard VR controllers, the Bracer adds on 6DoF hand tracking while keeping the user's hands free to interact with material objects.

Antilatency Extension Board

Pcb for easy using Antilatency Hardware Extension Interface

Extension Module

Модуль для подключения к HMD Radio Socket дополнительных компонентов с помощью Antilatency Hardware Extension Interface. Используя модуль, можно считывать внешние триггеры с кнопок или, например, подключать дополнительные микросхемы.

HMD Radio Socket

The HMD Radio Socket is a wireless Socket that supports both data reception from wireless devices via the Antilatency Radio Protocol and tracking data transmission when placed on the objects being tracked.

Pico G2 Socket

The Pico G2 Socket is a wireless Socket designed to track the PicoG2 headset and receive data from wireless devices via the Antilatency Radio Protocol.

SocketReferenceDesign

A development board for Socket with an Alt connector. You can use it as a base to custom your own Socket according to our recommedations.

Socket USB

Лёгкий и маленький Socket без поддержки Antilatency Radio Protocol идёт в комплекте с Alt.

Tag

The Tag is a wireless Socket that can be attached to an object or a part of the human body to track them.
The Tag does not track its own position, but relays data from the Alt fitted into it.
The Tag can track material objects and custom controllers, for example dummy guns.

Напольная зона трекинга 10м²

Зона трекинга состоит из специальных лент с ИК-маркерами, мягкой плитки и коннекторов. Alt определяет своё положение в пространстве, анализируя расположение маркеров.
