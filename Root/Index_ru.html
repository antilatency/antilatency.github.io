Оборудование

Alt
Alt это оптико-инерциальный модуль трекинга. Устройство размещается на объектах отслеживания и определяет свое положение в пространстве относительно инфракрасных маркеров. Alt поставляется в комплекте с проводным SocketUsb.
Bracer
Bracer - это wireless Socket для отслеживания рук пользователя, спроектированный для удобного размещения на руке. В отличие от стандартных VR-контроллеров, добавляя 6DoF отслеживание рук, Bracer оставляет руки пользователя свободными для взаимодействия с физическими предметами.
Antilatency Extension Board
Плата для удобного использования Antilatency Hardware Extension Interface
Pico G2 Socket
Pico G2 Socket - это wireless Socket, спроектированный под шлем PicoG2 для трекинга шлема и приема данных от беспроводных устройств по Antilatency Radio Protocol.
SocketReferenceDesign
Отладочная плата для разработчиков Socket, на основе которой можно собрать собственный Socket. Мы разработали общие рекомендации к параметрам самой платы и к дополнительным элементам, которые вы можете на неё поставить.
HMD Radio Socket
HMD Radio Socket - это wireless Socket, который поддерживает как прием данных от беспроводных устройств по Antilatency Radio Protocol, так и передачу данных трекинга при размещении на объектах отслеживания.
Tag
Tag - это беспроводной Socket, который можно закрепить на объекте или на части тела человека для их отслеживания. Он передаёт данные с подключённого к нему Alt. Используя Tag, вы можете отслеживать положение физических предметов и кастомных контроллеров, например, макетов оружия.
Напольная зона трекинга 10m²
Напольная зона трекинга собирается из специальных лент с ИК-маркерами. Ленты с ИК-маркерами фиксируются с помощью напольных плиток. Помимо лент с маркерами в комплект входят коннекторы и соединительные ленты для соединения лент между собой.
