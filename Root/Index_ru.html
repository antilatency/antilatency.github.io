Оборудование

Alt

Оптико-инерциальный модуль трекинга небольшого размера имеет угол обзора в 240 градусов. С его помощью вы можете отслеживать положение любого физического объекта. Alt поставляется в комплекте с Socket USB.

Bracer

Небольшой лёгкий Socket с поддержкой 6DoF, спроектированный для отслеживания положения рук. Благодаря эргономичному дизайну, можно свободно взаимодействовать с физическими предметами во время трекинга.

Antilatency Extension Board

Плата для подключения к HMD Radio Socket внешних компонентов, например, кнопок или LED. С её помощью вы можете как считывать внешние триггеры, так и управлять другими устройствами, например, если нужно настроить виброотклик.

Extension Module

Модуль для подключения к HMD Radio Socket дополнительных компонентов с помощью Antilatency Hardware Extension Interface. Используя модуль, можно считывать внешние триггеры с кнопок или, например, управлять виброоткликом устройства через встроенный вибромоторчик.

HMD Radio Socket

Маленький и лёгкий Socket. HMD Radio Socket может работать и как приёмник, и как передатчик. Более того, через Antilatency Extension Board устройство может считывать внешние физические триггеры, например, нажание кнопки.

Pico G2 Socket

Беспроводной Socket, спроектированный специально для отслеживания шлема PicoG2. Благодаря дополнительному разъёму USB type-C Female, можно заряжать VR шлем через PicoG2Socket.

SocketReferenceDesign

Отладочная плата для разработчиков Socket, на основе которой можно собрать собственный Socket. Мы разработали рекомендации к параметрам платы и её дополнительных компонентов.

Socket USB

Лёгкий и маленький Socket без поддержки Antilatency Radio Protocol идёт в комплекте с Alt.

Tag

Беспроводной Socket, который можно удобно разместить даже на небольшом объекте из-за малых габаритов и веса. Кроме того, Tag имеет встроенный аккумулятор и остаётся полностью функционален во время зарядки.

Напольная зона трекинга 10м²

Зона трекинга состоит из специальных лент с ИК-маркерами, мягкой плитки и коннекторов. Alt определяет своё положение в пространстве, анализируя расположение маркеров.
