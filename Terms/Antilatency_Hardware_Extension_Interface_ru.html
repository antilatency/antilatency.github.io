Antilatency Hardware Extension Interface

Реализаци доступа к GPIO сокета через UsbTypeC разъём с помощью Antilatency Extension Board и Antilatency Hardware Extension Interface Library

Поддерживаемые сокеты

  • HMD Radio Socket(hardware version 2.0.0) - режим UsbRadioSocket и RadioSocket.

Схема подключение

Используется Antilatency Extension Board 2.0.0.
Сокет подключается к Antilatency Extension Board при помощи Usb 3.1 Type-C кабеля, во второй разъём подключается питание с помощью любого Type-C кабеля. Antilatency Extension Board позволяет подключать различные кнопки, концевики, потенциометры, светодиоды, вибромоторчики и т.д.
