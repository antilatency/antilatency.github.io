Antilatency Hardware Extension Interface

Реализаци доступа к GPIO сокета через UsbTypeC разъём с помощью Antilatency Extension Board и Antilatency Hardware Extension Interface Library

Поддерживаемые сокеты

  • HMD Radio Socket(hardware version 2.0.0) - режим UsbRadioSocket и RadioSocket.

Подключение

Используется Antilatency Extension Board 2.0.0.
  • Сокет подключается к Antilatency Extension Board при помощи Usb 3.1 Type-C кабеля(важно именно 3.1, так как нужны все 4 дифф. пары в кабели). Подключить необходимо одинаковой ориентацией коннекторов для того, чтобы соединить сокет и плату расширения один к одному без перекрещивания сигнальных линий. Существуют кабели, внутри которых, уже есть перекрещивание, такие необходимо подключать наоборот - разной ориентацией.
  • Питание подключается к Antilatency Extension Board с помощью любого Type-C кабеля.
Antilatency Extension Board не симметрична, то есть подключать нужно с правильной стороны(как на схеме). Синие коннекторы стоят со стороны разъема для питания.
