Antilatency Hardware Extension Interface

Accessing the GPIO of a socket through a USB type C connector with Antilatency Extension Board and Antilatency Hardware Extension Interface Library

Currently supported

  • SocketUsbRadio(hardware version 2.0.0) - UsbRadioSocket and RadioSocket modes.

Connection

Antilatency Extension Board 2.0.0 is used.
  • Connect the socket to Antilatency Extension Board using a USB 3.1 Type-C cable(3.1 is important as all 4 differential pairs are needed for it to work). It is necessary to connect the socket and the extension board with the same orientation of the connectors to prevent crossing the signal lines. There are cables which already have a crossover of signal lines inside. You need to connect such cables the other way around, that is, upside down.
  • Connect Antilatency Extension Board to a power supply with a Type-C cable.
Antilatency Extension Board is not symmetrical, You should connect the board from the right side. The power connector is placed below the blue connectors (see the scheme above).
