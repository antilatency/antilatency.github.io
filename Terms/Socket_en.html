Socket

Socket это устройство (hardware), имеющее в своем составе разъем для подключения трекера Alt.
Сокеты обеспечивают connectivity для трекера, а также могут обладать собственной функциональностью.
В частности, Alt сам по себе не обладает радиомодулем, но если вставить Alt в сокет с радиомодулем (например, Tag), то можно передавать данные трекинга по радиоканалу.
Аналогично, SocketUsb может быть использован для передачи данных с трекера по USB-проводу на Host.
Помимо передачи данных с Alt, Socket может обладать собственной функциональностью. Например, Bracer обладает функциональностью по распознаванию касания touch-панели и на хост передаются как данные трекинга так и события касания touch-панели. Кроме того, Bracer имеет в своем составе модуль вибро отклика, который может управляться из приложения пользователя.

Создание собственных устройств с поддержкой трекера

Существует открытый дизайн сокета (SocketReferenceDesign). Пользователи могут создавать собственные устройства с поддержкой трекера Alt, реализуя спецификацию из SocketReferenceDesign.
