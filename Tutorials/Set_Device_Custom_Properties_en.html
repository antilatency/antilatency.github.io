Конфигурация пользовательских свойств

Устройства Antilatency имеют ряд системных и пользовательских свойств. Управляя пользовательскими свойствами, вы можете настроить устройство для конкретной задачи.
Все свойства устройств Antilatency отображаются во вкладке Device Network приложения AntilatencyService. Чтобы их увидеть, подключите устройство к Host (например, к компьютеру).

Во вкладке Device Network вы можете:
  • отредактировать пользовательское свойство
  • добавить новое свойство
  • удалить пользовательское свойство

Как отредактировать пользовательское свойство

Представим, что у вас есть два Bracer: для левой руки и для правой. Чтобы определить, какое из устройств за какой рукой будет закреплено, нужно назначить для каждого Bracer свойство Tag со значениями LeftHand и RightHand соответственно.
Шаг 1. В списке справа выберите свойство Tag и пропишите в нём LeftHand. Обратите внимание, значок свойства изменился.
Шаг 2. Чтобы отправить свойство на устройство, нажмите на иконку в правом нижнем углу и выберите опцию Send settings to device. Когда процесс загрузки закончится, значок свойства изменится на стандартный.
Если в процессе работы со свойствами вам нужно снова загрузить данные с устройства, воспользуйтесь опцией Reload settings from device.

Как создать новое пользовательское свойство

Шаг 1. Чтобы добавить новое свойство для устройства, нажмите на значок +.
Шаг 2. Присвойте свойству имя и значение.
Шаг 3. С помощью опции Send settings to device запишите новое свойство на устройство. Когда процесс загрузки закончится, значок свойства изменится на стандартный.

Как удалить пользовательское свойство

Шаг 1. Чтобы убрать какие-либо свойство, нажмите на значок x. Если вы передумали, нажмите на значок повторно.
Шаг 2. Используйте опцию Send settings to device, чтобы отправить изменённый набор свойств на устройство.
